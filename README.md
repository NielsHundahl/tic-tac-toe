# Tic-Tac-Toe 🎮

A simple two-player Tic-Tac-Toe game built with HTML, CSS, and JavaScript.

## How to play

1. Open `index.html` in your browser — no installation needed!
2. Players take turns clicking cells to place their mark (X or O)
3. First player to get 3 in a row wins
4. Click **Restart Game** to play again
